FORMULARIO

lunghezza d'onda = velocita[m/s] / frequenza[s-1]

frequenza media(bit/s) / frequenza di picco = fattore di burstiness | OPPURE(non so perchè) | frequenza di picco / frequenza media = fattore di burstiness

ritardo totale di trasferimento = tempo di trasferimento + tempo di propagazione

ritardo totale di trasferimento[nodo X e nodo Y] =
tempo di trasferimento[nodo X] + tempo di propagazione[nodo X] =
numero di bit / frequenza di trasmissione[nodo X] + tempo di propagazione[nodo X]

Tx-y = Ttx + tx = nbit / Fx + tx

ritardo totale di trasferimento[nodo X e nodo Y e nodo Z] =
tempo di trasferimento[nodo X] + tempo di propagazione[nodo X] + tempo di elaborazione[nodo Y] + tempo di trasferimento[nodo Y] + tempo di propagazione[nodo Y] =
numero di bit / frequenza di trasmissione[nodo X] + tempo di propagazione[nodo X] + tempo di elaborazione[nodo Y] + numero di bit / frequenza di trasmissione[nodo Y] + tempo di propagazione[nodo Y]

Tx-z = Ttx + tx + Tpy + Tty + ty = nbit / Fx + tx + tpy + nbit / Fy + ty

Erlang = throughput / capacita di rete

tasso di errore sul bit = numero di bit ricevuti / numero di bit trasmessi

tasso di perdita di UI = numero di UI perse o consegnate errate / numero di UI trasmesse

tempo di ritardo = tempo di trasmissione + tempo di propagazione + ( ∑ i = 1 N Nodi tempo di elaborazione + tempo di memorizzazione + tempo di trasmissione + tempo di propagazione)

valore medio dell'intensità del traffico in ingresso = numero di flussi che costituiscono traffico offerto(in ingresso) * sorgente con valore di burstiness * frequenza di picco(sorgente)[bit/s]

fattore di concentrazione = numero di flussi che costituiscono traffico offerto(in ingresso) * frequenza di picco(sorgente)[bit/s] / flusso in uscita[bit/s]
